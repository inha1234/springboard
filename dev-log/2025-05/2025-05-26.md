부모 id 값을 설정 하지못하는 것으로 인한 오류

### ✅ 문제 해결
- 못하는중

### 💡 느낀 점
- 하려고 한 방법이 comment를 따로 생성하면 id 값이 자동으로 생기지 않을까
하는 생각으로 static으로 comment 값을 생성해보았지만 id 값이 null로 잡혀서
좀더 생각해보고 고민해봐야할거같음
